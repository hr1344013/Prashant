CSS Selectors

ApplyingCSS to different Parts of HTML

1. The element selector targets elements based on their HTML tag name .

  1. Class selectors targets elements based on the value of the a class attribute.
  2. ID Selectors targets elements based on the value of ID attribute.
  3. Attribute selectors target the elemnts based on their values and attribute.
  4. The universal selectors target all the elements
